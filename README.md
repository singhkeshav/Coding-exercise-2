## Requirements

* Node 8+
* Git
* Node CLI 
* Mocha* (using npm install -g mocha)

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/singhkeshav/Coding-exercise-2.git
cd Coding-exercise-2
```

```bash
npm install
npm run test...

```
